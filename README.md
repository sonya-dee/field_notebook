## Getting Started

To run this app locally:

```bash
# 1. Clone the repository
git clone https://github.com/sonya-dee/field_notebook.git

# 2. Navigate into the project folder
cd field_notebook

# 3. Install dependencies
npm install

# 4. Start the development server
npm start
```
